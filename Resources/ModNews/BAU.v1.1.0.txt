We are thrilled to announce the release of <b>BetterAmongUs</b> v1.1.0
#
BetterAmongUs comes packed with a variety of features designed to improve your gameplay experience:
#
<size=150%>Change Log</size>
#
<b>Client</b>
New: Added new intro on boot up
New: Added ability to skip intro by clicking the left or right Mouse button
New: Added Better Option to have chat always visible
Fixed: All players appearing in the role intro
#
<b>Lobby Settings</b>
New: Added the ability to bypass number restrictions in lobby settings
New: Added Better Settings tab with new settings
New: Game Settings is now the default tab when opening lobby settings panel
New: Lobby Settings panel is now available for Non-Host but only Better Settings is editable
Hide And Seek
New: Added Better Setting to set imposter amount
Requires Better Role Algorithm
#
<b>Anti-Cheat</b>
Added list of Better Settings
1. When a player is caught cheating
2. Allow invalid friendCodes
3. Use Ban Player List
4. Use Ban Name List
5. Use Ban Word List
6. Detected player level = or >
7. Detect Cheat Clients
8. Detect Invalid RPCs
9. Detect Invalid Sabotages // Needs testing!
